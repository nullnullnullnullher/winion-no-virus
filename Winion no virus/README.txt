# 🧬 Winion Life Simulator

Уютный и безопасный 2D-симулятор искусственной жизни виртуальных питомцев (Ионов) с продвинутой системой цифрового мозга (1000 нейронных секторов), кулинарным крафтом и интерактивным обучением.

---

## 🚀 Как запустить игру

1. Установите библиотеку **Pygame-CE** (современная и стабильная версия Pygame):
   ```bash
   pip install pygame-ce
   ```
2. Запустите главный файл игры из папки проекта:
   ```bash
   python main.py
   ```

---

## 📁 Структура файлов проекта

Для корректной работы игры и отображения покадровой анимации, структура ваших папок должна выглядеть так:
```text
📂 Ваша_Папка_Игры/
 ├── 📄 main.py (Главный код игры)
 └── 📂 ion_animate/
      ├── 🖼️ egg_ion.png   (Текстура яйца)
      ├── 🖼️ ion1.png      (Кадр шага 1)
      ├── 🖼️ ion2.png      (Кадр шага 2)
      └── 🖼️ ion_dead.png  (Кадр погибшего питомца)
```

## 🕹️ Игровой процесс и Управление

* **`ESC`** — Мгновенный безопасный выход из игры с сохранением всех данных.
* **`ПКМ` (Правая кнопка мыши)** — Открывает меню создания ресурсов на ковре (Яблоко, Вода, Мука, Ягода).
* **`ЛКМ` (Левая кнопка мыши)** — Взаимодействие:
  * Клик по предмету — перенести ингредиент на разделочный стол.
  * Клик по Иону — открыть меню воспитания (**Ласка** / **Наказать**).
  * Клик по кухонному столу — запустить процесс готовки.

### 🧠 Умная система жестов (Обучение Иона)
Если вы создали еду на полу, вы можете управлять поведением Иона с помощью кастомного прицела-курсора:
* **Жест «НЕЛЬЗЯ»:** Быстро водите прицелом влево-вправо над едой. Предмет подсветится красным, а Ион запишет запрет в сектор мозга №400. Если Ион не сильно голоден, он послушается. Но если голод превысит 75%, инстинкт выживания победит страх, и он украдет еду!
* **Жест «МОЖНО»:** Покрутите курсором круги вокруг еды, чтобы снять запрет и позвать Иона кушать.

## 💾 Система сохранений

Игра полностью безопасна и не использует скрытые процессы. 
* **Характеристики и ДНК:** Данные Ионов автоматически записываются в текстовые файлы (`stats.txt` и `brain.txt`) по адресу: `Этот компьютер ➔ Документы ➔ WinionLife_Saves`.
* **Умный Таймер:** Время созревания яйца (5 минут) привязано к системным часам Windows и сохраняется в файл `egg_timer.txt`. Если вы выключите игру, таймер продолжит идти в реальном мире!
